Using LLMs to write emails

LLMs can help you write emails quickly and take on additional information to change the tone, length and translate the emails.

Here are some things that you can try out using the LLM you are using.

Writing an email

To create an email you can give a simple prompt to the LLM. For example

# Write a response to an email request in a formal tone. I don’t want to present at that conference.

The LLM will generate an email template for you. You can then ask it to make improvements if needed.

Changing tone and translating emails.

Here is another example of prompts where the LLM generates and email and then you can ask it to change the words, tone, length, language and writing style. Try the prompts below one after the other in the LLM.

# 1. Write an email to request that Peter send me the data he has promised. I sent two letters 
    already. Use a nice tone.
  2. Rewrite in other words.
  3. Sharpen the tone.
  4. Shorten.
  5. Translate to Swedish.
  6. use the english version and write the request as a poem in old english.

Another example of prompt where user writes an email in anger but wants the tone to be nicer is shown below

# Subject: Data Request

  Dear Peter,
  
  The data you sent is not formatted correctly. The analysis using the data does not make sense at all. This is very unprofessional and below expectations for the role you have. I am not happy at all.
     
  Regards,
    
  [Your Name]
    
  Rewrite my email, correct language, make the tone a bit nicer.

Try out different changes to the emails and see how the LLM responds.
